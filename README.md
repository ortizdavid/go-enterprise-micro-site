### O que é o Go Enterprise Micro?
É um chassis industrial para Go (Golang) desenhado para alta performance e escalabilidade. 
- **Clean Architecture**
- **CQRS Primordial**
- **Outbox Pattern**
- **Automated Scaffolding**
